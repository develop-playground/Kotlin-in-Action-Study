## Project

### 편의점 포스기 구현하기

- 무엇을 구현하면 좋을까? : 메뉴 CRUD, 한 주문에 고른 메뉴의 가격 합, 하루 매출 중에 카드 결제,  현금 결제 구분하여 가격 합 출력, `enum`으로 현금/카드를 구분해서 출력, 시작, 종료, 프로세스 선택하기, 결제 방식이나 상태 고르는 기능
